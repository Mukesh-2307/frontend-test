# Project Setup Guide

Follow the steps below to run the project locally:

### Step 1:
Fork this repository or download the ZIP file and extract it.

### step 2:
Open the project folder in your preferred code editor.

### step 3:
Open the terminal (Shortcut: Shift + `).

### step 4:
Run the following command to install dependencies:

```
npm install
```

Wait for the installation to complete.

### step 5:
Start the development server:

```
npm run dev
```

### step 6:
In the terminal output, hover over the local server URL and press Ctrl + Left Click to open it in your browser.
